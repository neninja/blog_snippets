Repetição de comandos terminal

15/02/2023
  • Edite este post
  • Histórico de edição

Diversas vezes preciso ficar repetindo um comando, seja algum tipo de ping ou teste. Ao invés de usar ⬆enter é possível utilizar somente enter com:

while $@; read line; do true; done;

Caso seja algo repetitivo, recomendo a criação de uma função no seu ~/.bashrc

# wrl echo "a"
wrl(){
  while $@; read line; do true; done;
}
